uploadFromFile( )

Description

Uploads an audio file to the Wix Media Platform. Accepts callbacks, or returns a promise. If callbacks are not supplied, a Promise is returned

Parameters

path string The local path to the audio file
success (optional) UploadStatus An optional callback triggered on success

Return Value

AudioMedia[ ] A promise that will yield an upload.data.AudioMedia object, or null if using callbacks
